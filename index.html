Java's Most Widely-Used Cache

Ehcache is an open source, standards-based cache for boosting performance, offloading your database, and simplifying scalability. It's the most widely-used Java-based cache because it's robust, proven, and full-featured. Ehcache scales from in-process, with one or more nodes, all the way to mixed in-process/out-of-process configurations with terabyte-sized caches.

Ehcache v3: API reboot & javax.cache driven

Ehcache is one of the most feature rich caching APIs out there. Yet it's been growing organically, all founded on the very early API as designed pre-Java5. We've learned a lot in the meantime and there have been compromises made along the way. But "fixing" these isn't an easy task, often even impossible without breaking backward compatibility. In the meantime, while some caching solutions took very different approaches on it all, the expert group on JSR-107, Terracotta included, put great efforts in trying to come up with a standard API. We felt the new major version of the API should be based on the standardized API. JSR-107 serves as the basis for the new API but Ehcache 3.0 extends the specification in many aspects.

Ehcache 3 : Milestone 1 reached - With Offheap Storage

With offheap storage committed, Milestone 1 of Ehcache 3 has been announced.
"Official" builds of m1 are available on the project's GitHub release page or from Maven Central - see below.

Getting started

You can introducing caching to your Java application quiet easily with Ehcache. Wheter using its new poweful API or using the javax.cache API as defined in the JSR-107 specification.

Using the core Ehcache v3 API

While new Ehcache v3 is still in heavy developement, you can give it a test ride today! We've working hard on bringing all the featureset from the previous releases to this new version. The core API is pretty much solid already and should give you a feel for what's coming...

Downloading the jars

You can download the jar directly from github or getting it from Maven central: 
  <dependency>
    <groupId>org.ehcache</groupId>
    <artifactId>ehcache</artifactId>
    <version>3.0.0.m1</version>
  </dependency>

Coding to the Ehcache 3 API


  CacheManager cacheManager = newCacheManagerBuilder()
             .withCache("preConfigured", newCacheConfigurationBuilder().buildConfig(Long.class, String.class))
             .build();

  Cache<Long, String> preConfigured
      = cacheManager.getCache("preConfigured", Long.class, String.class);

  Cache<Long, String> myCache = cacheManager.createCache("myCache",
      newCacheConfigurationBuilder().buildConfig(Long.class, String.class));

  myCache.put(1L, "da one!");
  String value = myCache.get(1L);

  cacheManager.close();

Downloading the jars
