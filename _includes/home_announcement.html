Ehcache 3 : Milestone 5 reached - All features are in!

Milestone 5 comes with over nearly 20 issues closed and includes new features, such as:
  • Support for byte sizing on heap,
  • Reworked cache event listener system,
  • Many improvements in configuration builders,
  • Alignment at the XML configuration level to offer better consistency,
  • See all closed issues
"Official" builds of m5 are available on the project's GitHub release page or from Maven Central - see below.
